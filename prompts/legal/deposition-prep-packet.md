# Deposition Prep Packet

**Category:** `legal`

**Apps Involved:** Word, PowerPoint

**Complexity:** Simple (1-2 apps)

### Description

Compiles a deposition preparation packet for a witness, including background facts, likely questions, and key document references.

### The Prompt

```
Using the case summary in [/CASE_SUMMARY] and the witness profile in [/WITNESS_PROFILE], create a deposition prep packet in Word. Include: (1) witness background summary, (2) key facts the witness is expected to address, (3) 20 likely deposition questions with suggested response guidance, (4) documents the witness should review before the deposition with a one-sentence description of each. Format for easy reading during a prep session. Save to [/MATTER_FOLDER/].
```

### Expected Outcome

A formatted Word deposition prep packet ready for an attorney-witness prep session.

### Tips & Variations

- Maintain a standard case summary file per matter in SharePoint as the source of truth for prompts like this.
- Variation: generate a shorter "cheat sheet" version as a one-pager in PowerPoint for the witness to take to the deposition.
