# Contributing

1. **Use the template.** Copy [TEMPLATE.md](TEMPLATE.md) and fill it in.
2. **One prompt per file.** Name it descriptively: `weekly-email-digest.md`, not `prompt1.md`.
3. **Test before submitting.** Only share prompts you've actually run in Cowork.
4. **Keep prompts self-contained.** Don't assume specific file names or folder structures in someone else's tenant.
5. **No sensitive data.** Redact company names, emails, and PII from examples.

## Pull Request Process

- Place your file in the correct `prompts/<category>/` folder.
- PR title format: `[category] Prompt title`
- Brief description of what the prompt does and which apps it touches.
