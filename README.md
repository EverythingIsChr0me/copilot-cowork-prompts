# Microsoft 365 Copilot Cowork Prompts

A community collection of practical [Cowork](https://support.microsoft.com/en-us/topic/get-started-with-cowork-in-microsoft-365-copilot-0f498cf3-fa48-4708-94a4-ef1530757897) prompts for Microsoft 365 Copilot — organized by industry and use case.

## What is Cowork?

Cowork is a capability in Microsoft 365 Copilot that lets you delegate multi-step tasks to an AI agent that works autonomously across your M365 apps (Word, Excel, PowerPoint, Outlook, Teams, etc.) and reports back when done.
