# List All Skills

**Category:** `personal-productivity`

**Apps Involved:** Cowork

**Complexity:** Simple (1-2 apps)

### Description

Asks Cowork to enumerate every skill available to it — both built-in/native skills and any custom skills that have been added to the workspace.

### The Prompt

```
List every skill you currently have access to.

Separate them into two groups: Native Skills (built into Cowork) and Custom Skills (added to this workspace).

For each skill, include the name and a one-sentence description of what it does.
```

### Expected Outcome

A two-section list clearly distinguishing native Cowork skills from custom skills, each with a name and short description.

### Tips & Variations

- Add `Return as a markdown table with columns: Skill Name | Type | Description.` for a cleaner format.
- To focus on just one group: replace the first sentence with `List only my custom skills.`
- Useful as a discovery step before chaining skills in a larger workflow.
